OPERATOR: Polarimetric-Parameters
----------------------------------------
Usage:
  gpt Polarimetric-Parameters [options] 

Description:
  Compute general polarimetric parameters


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PoutputBMI=<boolean>               Output BMI
                                      Default value is 'false'.
  -PoutputCSI=<boolean>               Output CSI
                                      Default value is 'false'.
  -PoutputHHHVRatio=<boolean>         Output Cross-Pol HH/HV
                                      Default value is 'false'.
  -PoutputHHVVRatio=<boolean>         Output Co-Pol HH/VV
                                      Default value is 'false'.
  -PoutputITI=<boolean>               Output ITI
                                      Default value is 'false'.
  -PoutputPedestalHeight=<boolean>    Output pedestal height
                                      Default value is 'false'.
  -PoutputRFDI=<boolean>              Output RFDI
                                      Default value is 'false'.
  -PoutputRVI=<boolean>               Output RVI
                                      Default value is 'false'.
  -PoutputSpan=<boolean>              Output Span
                                      Default value is 'true'.
  -PoutputVSI=<boolean>               Output VSI
                                      Default value is 'false'.
  -PoutputVVVHRatio=<boolean>         Output Cross-Pol VV/VH
                                      Default value is 'false'.
  -PuseMeanMatrix=<boolean>           Use mean coherency or covariance matrix
                                      Default value is 'true'.
  -PwindowSizeXStr=<string>           Sets parameter 'windowSizeXStr' to <string>.
                                      Value must be one of '3', '5', '7', '9', '11', '13', '15', '17', '19'.
                                      Default value is '5'.
  -PwindowSizeYStr=<string>           Sets parameter 'windowSizeYStr' to <string>.
                                      Value must be one of '3', '5', '7', '9', '11', '13', '15', '17', '19'.
                                      Default value is '5'.
