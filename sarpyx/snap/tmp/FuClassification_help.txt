OPERATOR: FuClassification
----------------------------------------
Usage:
  gpt FuClassification [options] 

Description:
  Colour classification based on the discrete Forel-Ule scale


Source Options:
  -Ssource=<file>    The source product providing reflectances.
                     This is a mandatory source.

Parameter Options:
  -PcopyAllSourceBands=<boolean>              Weather or not to copy all the bands to the target product from the source product.
                                              Default value is 'false'.
  -PincludeDominantLambda=<boolean>           Whether or not the dominant wavelength shall be derived from the hue angle
                                              Default value is 'false'.
  -PincludeIntermediateResults=<boolean>      Whether or not the intermediate results shall be written to the target output
                                              Default value is 'true'.
  -PinputIsIrradianceReflectance=<boolean>    If enabled, the source reflectances will be converted to radiance reflectances by dividing it by PI before passing to the algorithm.
                                              Default value is 'false'.
  -Pinstrument=<instrument>                   The instrument to compute FU for.
                                              Default value is 'AUTO_DETECT'.
  -PreflectanceNamePattern=<string>           The used reflectance band names must match the given pattern. Useful, if there is more then one spectrum in the product.
  -PvalidExpression=<string>                  An expression to filter which pixel are considered.
