OPERATOR: Double-Difference-Interferogram
----------------------------------------
Usage:
  gpt Double-Difference-Interferogram [options] 

Description:
  Compute double difference interferogram


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PcohWinSize=<string>          Sets parameter 'cohWinSize' to <string>.
                                 Value must be one of '3', '5', '9', '11'.
                                 Default value is '5'.
  -PoutputCoherence=<boolean>    Output coherence for overlapped area
                                 Default value is 'false'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Double-Difference-Interferogram</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <outputCoherence>boolean</outputCoherence>
        <cohWinSize>string</cohWinSize>
      </parameters>
    </node>
  </graph>
