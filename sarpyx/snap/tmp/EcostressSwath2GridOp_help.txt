OPERATOR: EcostressSwath2GridOp
----------------------------------------
Usage:
  gpt EcostressSwath2GridOp [options] 

Description:
  Performs ECOSTRESS Swath to Grid conversion to a specified CRS and radiance to brightness temperature conversion, using the algorithm from 'ECOSTRESS Swath to Grid Conversion Script' by LP DAAC (https://git.earthdata.nasa.gov/projects/LPDUR/repos/ecostress_swath2grid/).


Source Options:
  -SecoProduct=<file>    The source product to be converted from Swath to Grid.
                         Valid product types must match 'ECOSTRESS-.*'.
                         This is a mandatory source.
  -SgeoProduct=<file>    The source GEO product from which the geocoding will be extracted.
                         Valid product types must match 'ECOSTRESS-L1B-GEO'.
                         This is a mandatory source.

Parameter Options:
  -PcrsIN=<string>      The projection desired for the output.
                        Value must be one of 'GEO', 'UTM'.
                        Default value is 'GEO'.
  -Pr2b=<boolean>       The optional argument to convert radiance to brightness temperature for the L1B products
                        Default value is 'false'.
  -PutmZone=<string>    The UTM zone (EPSG Code) desired for all outputs - only required if needed to override default UTM zone which is assigned based on the center location for each ECOSTRESS granule.
                        Default value is 'auto_lookup'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>EcostressSwath2GridOp</operator>
      <sources>

        <geoProduct>${geoProduct}</geoProduct>
      </sources>
      <parameters>
        <crsIN>string</crsIN>
        <utmZone>string</utmZone>
        <r2b>boolean</r2b>
      </parameters>
    </node>
  </graph>
