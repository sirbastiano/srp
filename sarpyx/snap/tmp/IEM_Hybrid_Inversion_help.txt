OPERATOR: IEM-Hybrid-Inversion
----------------------------------------
Usage:
  gpt IEM-Hybrid-Inversion [options] 

Description:
  Performs IEM inversion using Hybrid approach


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -PdoRemainingOutliersFilter=<boolean>    Replace remaining outlier with neighbours's average
                                           Default value is 'true'.
  -PlutFile=<file>                         Sets parameter 'lutFile' to <file>.
  -PM=<integer>                            Length (pixels) of side of square neighbourhood (M)
                                           Default value is '5'.
  -PN=<integer>                            # closest sigma match from LUT search
                                           Default value is '5'.
  -PoutputCL=<boolean>                     Optional cl in output
                                           Default value is 'false'.
  -PoutputRMS=<boolean>                    Optional rms in output
                                           Default value is 'false'.
  -PthresholdRDC=<double>                  RDC deviation threshold
                                           Default value is '0.5'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>IEM-Hybrid-Inversion</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <N>integer</N>
        <M>integer</M>
        <doRemainingOutliersFilter>boolean</doRemainingOutliersFilter>
        <lutFile>file</lutFile>
        <outputRMS>boolean</outputRMS>
        <outputCL>boolean</outputCL>
        <thresholdRDC>double</thresholdRDC>
      </parameters>
    </node>
  </graph>
