OPERATOR: Cross-Correlation
----------------------------------------
Usage:
  gpt Cross-Correlation [options] 

Description:
  Automatic Selection of Ground Control Points


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -PapplyFineRegistration=<boolean>              Sets parameter 'applyFineRegistration' to <boolean>.
                                                 Default value is 'false'.
  -PcoarseRegistrationWindowHeight=<string>      Sets parameter 'coarseRegistrationWindowHeight' to <string>.
                                                 Value must be one of '32', '64', '128', '256', '512', '1024', '2048'.
                                                 Default value is '128'.
  -PcoarseRegistrationWindowWidth=<string>       Sets parameter 'coarseRegistrationWindowWidth' to <string>.
                                                 Value must be one of '32', '64', '128', '256', '512', '1024', '2048'.
                                                 Default value is '128'.
  -PcoherenceThreshold=<double>                  The coherence threshold
                                                 Valid interval is (0, *).
                                                 Default value is '0.6'.
  -PcoherenceWindowSize=<int>                    The coherence window size
                                                 Valid interval is (1, 16].
                                                 Default value is '3'.
  -PcolumnInterpFactor=<string>                  Sets parameter 'columnInterpFactor' to <string>.
                                                 Value must be one of '2', '4', '8', '16'.
                                                 Default value is '2'.
  -PcomputeOffset=<boolean>                      Sets parameter 'computeOffset' to <boolean>.
                                                 Default value is 'false'.
  -PfineRegistrationOversampling=<string>        Sets parameter 'fineRegistrationOversampling' to <string>.
                                                 Value must be one of '2', '4', '8', '16', '32', '64'.
                                                 Default value is '16'.
  -PfineRegistrationWindowAccAzimuth=<string>    Sets parameter 'fineRegistrationWindowAccAzimuth' to <string>.
                                                 Value must be one of '2', '4', '8', '16', '32', '64'.
                                                 Default value is '16'.
  -PfineRegistrationWindowAccRange=<string>      Sets parameter 'fineRegistrationWindowAccRange' to <string>.
                                                 Value must be one of '2', '4', '8', '16', '32', '64'.
                                                 Default value is '16'.
  -PfineRegistrationWindowHeight=<string>        Sets parameter 'fineRegistrationWindowHeight' to <string>.
                                                 Value must be one of '8', '16', '32', '64', '128', '256', '512'.
                                                 Default value is '32'.
  -PfineRegistrationWindowWidth=<string>         Sets parameter 'fineRegistrationWindowWidth' to <string>.
                                                 Value must be one of '8', '16', '32', '64', '128', '256', '512'.
                                                 Default value is '32'.
  -PgcpTolerance=<double>                        Tolerance in slave GCP validation check
                                                 Valid interval is (0, *).
                                                 Default value is '0.5'.
  -PinSAROptimized=<boolean>                     Sets parameter 'inSAROptimized' to <boolean>.
                                                 Default value is 'false'.
  -PmaxIteration=<int>                           The maximum number of iterations
                                                 Valid interval is (1, 10].
                                                 Default value is '10'.
  -PnumGCPtoGenerate=<int>                       The number of GCPs to use in a grid
                                                 Valid interval is (10, *).
                                                 Default value is '2000'.
  -PonlyGCPsOnLand=<boolean>                     Sets parameter 'onlyGCPsOnLand' to <boolean>.
                                                 Default value is 'false'.
  -ProwInterpFactor=<string>                     Sets parameter 'rowInterpFactor' to <string>.
                                                 Value must be one of '2', '4', '8', '16'.
                                                 Default value is '2'.
  -PuseSlidingWindow=<boolean>                   Use sliding window for coherence calculation
                                                 Default value is 'false'.
