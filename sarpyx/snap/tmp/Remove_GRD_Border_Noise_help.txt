OPERATOR: Remove-GRD-Border-Noise
----------------------------------------
Usage:
  gpt Remove-GRD-Border-Noise [options] 

Description:
  Mask no-value pixels for GRD product


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -PborderLimit=<int>                                   The border margin limit
                                                        Default value is '500'.
  -PselectedPolarisations=<string,string,string,...>    The list of polarisations
  -PtrimThreshold=<double>                              The trim threshold
                                                        Default value is '0.5'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Remove-GRD-Border-Noise</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <selectedPolarisations>string,string,string,...</selectedPolarisations>
        <borderLimit>int</borderLimit>
        <trimThreshold>double</trimThreshold>
      </parameters>
    </node>
  </graph>
