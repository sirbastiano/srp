OPERATOR: Flip
----------------------------------------
Usage:
  gpt Flip [options] 

Description:
  flips a product horizontal/vertical


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PflipType=<string>                         Sets parameter 'flipType' to <string>.
                                              Value must be one of 'Horizontal', 'Vertical', 'Horizontal and Vertical'.
                                              Default value is 'Vertical'.
  -PsourceBands=<string,string,string,...>    The list of source bands.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Flip</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <sourceBands>string,string,string,...</sourceBands>
        <flipType>string</flipType>
      </parameters>
    </node>
  </graph>
