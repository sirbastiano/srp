OPERATOR: BatchSnaphuUnwrapOp
----------------------------------------
Usage:
  gpt BatchSnaphuUnwrapOp [options] 

Description:
  Downloads and executes SNAPHU on a set of two or more interferograms.


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PsnaphuInstallLocation=<file>       Directory to install the SNAPHU binary to
  -PsnaphuProcessingLocation=<file>    Directory that the Snaphu Export operator wrote to

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>BatchSnaphuUnwrapOp</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <snaphuProcessingLocation>file</snaphuProcessingLocation>
        <snaphuInstallLocation>file</snaphuInstallLocation>
      </parameters>
    </node>
  </graph>
