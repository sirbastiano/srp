OPERATOR: OlciAnomalyFlagging
----------------------------------------
Usage:
  gpt OlciAnomalyFlagging [options] 

Description:
  Adds a flagging band indicating saturated pixels and altitude data overflows


Source Options:
  -Sl1bProduct=<file>    OLCI L1b or fully compatible product.
                         This is a mandatory source.

Parameter Options:
  -PwriteSlopeInformation=<boolean>    If set to true, the operator adds two bands containing the maximal spectral slope and the band index where the peak is observed.
                                       Default value is 'false'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>OlciAnomalyFlagging</operator>
      <sources>
        <l1bProduct>${l1bProduct}</l1bProduct>
      </sources>
      <parameters>
        <writeSlopeInformation>boolean</writeSlopeInformation>
      </parameters>
    </node>
  </graph>
