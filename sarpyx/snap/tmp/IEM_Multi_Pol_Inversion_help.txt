OPERATOR: IEM-Multi-Pol-Inversion
----------------------------------------
Usage:
  gpt IEM-Multi-Pol-Inversion [options] 

Description:
  Performs IEM inversion using Multi-polarization approach


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PdoRemainingOutliersFilter=<boolean>    Replace remaining outlier with neighbours's average
                                           Default value is 'true'.
  -PlutFile=<file>                         Sets parameter 'lutFile' to <file>.
  -PM=<integer>                            Length (pixels) of side of square neighbourhood (M)
                                           Default value is '5'.
  -PN=<integer>                            # closest sigma match from LUT search
                                           Default value is '5'.
  -PoutputRMS=<boolean>                    Optional rms in output
                                           Default value is 'false'.
  -PthresholdRDC=<double>                  RDC deviation threshold
                                           Default value is '0.5'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>IEM-Multi-Pol-Inversion</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <N>integer</N>
        <M>integer</M>
        <doRemainingOutliersFilter>boolean</doRemainingOutliersFilter>
        <lutFile>file</lutFile>
        <outputRMS>boolean</outputRMS>
        <thresholdRDC>double</thresholdRDC>
      </parameters>
    </node>
  </graph>
