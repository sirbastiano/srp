OPERATOR: HorizontalVerticalMotion
----------------------------------------
Usage:
  gpt HorizontalVerticalMotion [options] 

Description:
  Computation of Horizontal/Vertical Motion Components


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -PrefPixelX=<int>    X position for reference pixel
                       Default value is '0'.
  -PrefPixelY=<int>    Y position for reference pixel
                       Default value is '0'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>HorizontalVerticalMotion</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <refPixelX>int</refPixelX>
        <refPixelY>int</refPixelY>
      </parameters>
    </node>
  </graph>
