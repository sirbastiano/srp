OPERATOR: REACTIV-Change-Detection
----------------------------------------
Usage:
  gpt REACTIV-Change-Detection [options] 

Description:
  REACTIV Change Detection.


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PincludeSourceBands=<boolean>    Include source bands
                                    Default value is 'false'.
  -PmaskThreshold=<float>           Mask threshold
                                    Valid interval is (0, 1).
                                    Default value is '0.8'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>REACTIV-Change-Detection</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <maskThreshold>float</maskThreshold>
        <includeSourceBands>boolean</includeSourceBands>
      </parameters>
    </node>
  </graph>
