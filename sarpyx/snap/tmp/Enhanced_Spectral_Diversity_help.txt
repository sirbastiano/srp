OPERATOR: Enhanced-Spectral-Diversity
----------------------------------------
Usage:
  gpt Enhanced-Spectral-Diversity [options] 

Description:
  Estimate constant range and azimuth offsets for a stack of images


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PcohThreshold=<double>                The coherence threshold for outlier removal
                                         Valid interval is (0, 1].
                                         Default value is '0.3'.
  -PdoNotWriteTargetBands=<boolean>      Do not write target bands
                                         Default value is 'false'.
  -PesdEstimator=<string>                ESD estimator used for azimuth shift computation
                                         Value must be one of 'Average', 'Periodogram'.
                                         Default value is 'Periodogram'.
  -PfineWinAccAzimuth=<string>           Sets parameter 'fineWinAccAzimuth' to <string>.
                                         Value must be one of '2', '4', '8', '16', '32', '64'.
                                         Default value is '16'.
  -PfineWinAccRange=<string>             Sets parameter 'fineWinAccRange' to <string>.
                                         Value must be one of '2', '4', '8', '16', '32', '64'.
                                         Default value is '16'.
  -PfineWinHeightStr=<string>            Sets parameter 'fineWinHeightStr' to <string>.
                                         Value must be one of '32', '64', '128', '256', '512', '1024', '2048'.
                                         Default value is '512'.
  -PfineWinOversampling=<string>         Sets parameter 'fineWinOversampling' to <string>.
                                         Value must be one of '32', '64', '128', '256'.
                                         Default value is '128'.
  -PfineWinWidthStr=<string>             Sets parameter 'fineWinWidthStr' to <string>.
                                         Value must be one of '32', '64', '128', '256', '512', '1024', '2048'.
                                         Default value is '512'.
  -PintegrationMethod=<string>           Method used for integrating the shifts network.
                                         Value must be one of 'L1', 'L2', 'L1 and L2'.
                                         Default value is 'L1 and L2'.
  -PmaxTemporalBaseline=<int>            Maximum temporal baseline (in days or number of images depending on the Temporal baseline type) between pairs of images to construct the network. Any number < 1 will generate a network with all of the possible pairs.
                                         Default value is '4'.
  -PnumBlocksPerOverlap=<int>            The number of windows per overlap for ESD
                                         Valid interval is [1, 20].
                                         Default value is '10'.
  -PoverallAzimuthShift=<double>         The overall azimuth shift
                                         Default value is '0.0'.
  -PoverallRangeShift=<double>           The overall range shift
                                         Default value is '0.0'.
  -PtemporalBaselineType=<string>        Baseline type for building the integration network
                                         Value must be one of 'Number of images', 'Number of days'.
                                         Default value is 'Number of images'.
  -PuseSuppliedAzimuthShift=<boolean>    Use user supplied azimuth shift
                                         Default value is 'false'.
  -PuseSuppliedRangeShift=<boolean>      Use user supplied range shift
                                         Default value is 'false'.
  -PweightFunc=<string>                  Weight function of the coherence to use for azimuth shift estimation
                                         Value must be one of 'None', 'Linear', 'Quadratic', 'Inv Quadratic'.
                                         Default value is 'Inv Quadratic'.
  -PxCorrThreshold=<double>              The peak cross-correlation threshold
                                         Valid interval is (0, *).
                                         Default value is '0.1'.
