OPERATOR: Unmix
----------------------------------------
Usage:
  gpt Unmix [options] 

Description:
  Performs a linear spectral unmixing.


Source Options:
  -Ssource=<file>    The source product.
                     This is a mandatory source.

Parameter Options:
  -PabundanceBandNameSuffix=<string>          The suffix for the generated abundance band names (name = endmember + suffix).
                                              Pattern for valid values is '[a-zA-Z_0-9]*'.
                                              Default value is '_abundance'.
                                              This is a mandatory parameter.
  -PcomputeErrorBands=<boolean>               If 'true', error bands for all source bands will be generated.
                                              Default value is 'false'.
  -PendmemberFile=<file>                      A text file containing (additional) endmembers in a table. Wavelengths must be given in nanometers.
  -PerrorBandNameSuffix=<string>              The suffix for the generated error band names (name = source + suffix).
                                              Pattern for valid values is '[a-zA-Z_0-9]*'.
                                              Default value is '_error'.
                                              This is a mandatory parameter.
  -PminBandwidth=<double>                     Minimum spectral bandwidth used for endmember wavelength matching.
                                              Valid interval is (0,*).
                                              Default value is '10.0'.
                                              Parameter unit is 'nm'.
  -PsourceBands=<string,string,string,...>    The list of spectral bands providing the source spectrum.
  -PunmixingModelName=<string>                The unmixing model.
                                              Value must be one of 'Unconstrained LSU', 'Constrained LSU', 'Fully Constrained LSU'.
                                              Default value is 'Constrained LSU'.
