OPERATOR: Azimuth-Shift-Estimation-ESD
----------------------------------------
Usage:
  gpt Azimuth-Shift-Estimation-ESD [options] 

Description:
  Estimate azimuth offset for the whole image


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PcohThreshold=<double>        The coherence threshold for outlier removal
                                 Valid interval is (0, 1].
                                 Default value is '0.15'.
  -PnumBlocksPerOverlap=<int>    The number of windows per overlap for ESD
                                 Valid interval is [1, 20].
                                 Default value is '10'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Azimuth-Shift-Estimation-ESD</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <cohThreshold>double</cohThreshold>
        <numBlocksPerOverlap>int</numBlocksPerOverlap>
      </parameters>
    </node>
  </graph>
