OPERATOR: Faraday-Rotation-Correction
----------------------------------------
Usage:
  gpt Faraday-Rotation-Correction [options] 

Description:
  Perform Faraday-rotation correction for quad-pol product


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PwindowSize=<int>    The sliding window size
                        Valid interval is [1, 100].
                        Default value is '10'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Faraday-Rotation-Correction</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <windowSize>int</windowSize>
      </parameters>
    </node>
  </graph>
