OPERATOR: ALOS-Deskewing
----------------------------------------
Usage:
  gpt ALOS-Deskewing [options] 

Description:
  Deskewing ALOS product


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PdemName=<string>                          The digital elevation model.
                                              Default value is 'SRTM 3Sec'.
  -PsourceBands=<string,string,string,...>    The list of source bands.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>ALOS-Deskewing</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <sourceBands>string,string,string,...</sourceBands>
        <demName>string</demName>
      </parameters>
    </node>
  </graph>
