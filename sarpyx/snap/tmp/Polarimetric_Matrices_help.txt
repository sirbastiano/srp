OPERATOR: Polarimetric-Matrices
----------------------------------------
Usage:
  gpt Polarimetric-Matrices [options] 

Description:
  Generates covariance or coherency matrix for given product


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -Pmatrix=<string>    The covariance or coherency matrix
                       Value must be one of 'C2', 'C3', 'C4', 'T3', 'T4'.
                       Default value is 'T3'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Polarimetric-Matrices</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <matrix>string</matrix>
      </parameters>
    </node>
  </graph>
