OPERATOR: c2rcc.landsat8
----------------------------------------
Usage:
  gpt c2rcc.landsat8 [options] 

Description:
  Performs atmospheric correction and IOP retrieval with uncertainties on Landsat-8 L1 data products.


Source Options:
  -SncepEndProduct=<file>         The second product providing air pressure values for pressure interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.
  -SncepStartProduct=<file>       The first product providing air pressure values for pressure interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.
  -SsourceProduct=<file>          Landsat 8 source product.
                                  This is a mandatory source.
  -StomsomiEndProduct=<file>      The second product providing ozone values for ozone interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.
  -StomsomiStartProduct=<file>    The first product providing ozone values for ozone interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.

Parameter Options:
  -PalternativeNNPath=<string>                    Path to an alternative set of neuronal nets. Use this to replace the standard set of neuronal nets.
  -PatmosphericAuxDataPath=<string>               Path to the atmospheric auxiliary data directory. Use either this or the specified products on the I/O Parameters tab. If the auxiliary data is not available at this path, the data will automatically be downloaded.
  -PCHLexp=<double>                               Chlorophyll exponent ( CHL = iop_apig^CHLexp * CHLfac).
                                                  Default value is '1.04'.
  -PCHLfac=<double>                               Chlorophyll factor ( CHL = iop_apig^CHLexp * CHLfac).
                                                  Default value is '21.0'.
  -PdemName=<string>                              The digital elevation model.
                                                  Value must be one of 'Copernicus 90m Global DEM', 'Copernicus 30m Global DEM', 'GETASSE30'.
                                                  Default value is 'Copernicus 90m Global DEM'.
  -PderiveRwFromPathAndTransmittance=<boolean>    Alternative way of calculating water reflectance. Still experimental.
                                                  Default value is 'false'.
  -Pelevation=<double>                            Used as fallback if elevation could not be taken from DEM.
                                                  Valid interval is (0, 8500).
                                                  Default value is '0'.
                                                  Parameter unit is 'm'.
  -PnetSet=<string>                               Set of neuronal nets for algorithm.
                                                  Value must be one of 'C2RCC-Nets', 'C2X-Nets'.
                                                  Default value is 'C2RCC-Nets'.
  -PoutputAcReflectance=<boolean>                 Sets parameter 'outputAcReflectance' to <boolean>.
                                                  Default value is 'true'.
  -PoutputAsRrs=<boolean>                         Write remote sensing reflectances instead of water leaving reflectances.
                                                  Default value is 'false'.
  -PoutputKd=<boolean>                            Sets parameter 'outputKd' to <boolean>.
                                                  Default value is 'true'.
  -PoutputOos=<boolean>                           Sets parameter 'outputOos' to <boolean>.
                                                  Default value is 'false'.
  -PoutputRhown=<boolean>                         Sets parameter 'outputRhown' to <boolean>.
                                                  Default value is 'true'.
  -PoutputRpath=<boolean>                         Sets parameter 'outputRpath' to <boolean>.
                                                  Default value is 'false'.
  -PoutputRtoa=<boolean>                          Sets parameter 'outputRtoa' to <boolean>.
                                                  Default value is 'true'.
  -PoutputRtosaGc=<boolean>                       Sets parameter 'outputRtosaGc' to <boolean>.
                                                  Default value is 'false'.
  -PoutputRtosaGcAann=<boolean>                   Sets parameter 'outputRtosaGcAann' to <boolean>.
                                                  Default value is 'false'.
  -PoutputTdown=<boolean>                         Sets parameter 'outputTdown' to <boolean>.
                                                  Default value is 'false'.
  -PoutputTup=<boolean>                           Sets parameter 'outputTup' to <boolean>.
                                                  Default value is 'false'.
  -PoutputUncertainties=<boolean>                 Sets parameter 'outputUncertainties' to <boolean>.
                                                  Default value is 'true'.
  -Pozone=<double>                                The value used as ozone if not provided by auxiliary data.
                                                  Valid interval is (0, 1000).
                                                  Default value is '330'.
                                                  Parameter unit is 'DU'.
  -Ppress=<double>                                The surface air pressure at sea level if not provided by auxiliary data.
                                                  Valid interval is (800, 1040).
                                                  Default value is '1000'.
                                                  Parameter unit is 'hPa'.
  -Psalinity=<double>                             The value used as salinity for the scene.
                                                  Valid interval is (0.000028, 43).
                                                  Default value is '35.0'.
                                                  Parameter unit is 'PSU'.
  -Ptemperature=<double>                          The value used as temperature for the scene.
                                                  Valid interval is (0.000111, 36).
                                                  Default value is '15.0'.
                                                  Parameter unit is 'C'.
  -PthresholdAcReflecOos=<double>                 Threshold for out of scope of nn training dataset flag for atmospherically corrected reflectances
                                                  Default value is '0.1'.
  -PthresholdCloudTDown865=<double>               Threshold for cloud test based on downwelling transmittance @865
                                                  Default value is '0.955'.
  -PthresholdRtosaOOS=<double>                    Threshold for out of scope of nn training dataset flag for gas corrected top-of-atmosphere reflectances.
                                                  Default value is '0.05'.
  -PTSMexp=<double>                               TSM exponent (TSM = TSMfac * iop_btot^TSMexp).
                                                  Default value is '3.1'.
  -PTSMfac=<double>                               TSM factor (TSM = TSMfac * iop_btot^TSMexp).
                                                  Default value is '1.72'.
  -PvalidPixelExpression=<string>                 Defines the pixels which are valid for processing.
