OPERATOR: c2rcc.seawifs
----------------------------------------
Usage:
  gpt c2rcc.seawifs [options] 

Description:
  Performs atmospheric correction and IOP retrieval on SeaWifs L1C data products.


Source Options:
  -SncepEndProduct=<file>         The second product providing air pressure values for pressure interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.
  -SncepStartProduct=<file>       The first product providing air pressure values for pressure interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.
  -SsourceProduct=<file>          SeaWiFS L1C source product.
                                  This is a mandatory source.
  -StomsomiEndProduct=<file>      The second product providing ozone values for ozone interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.
  -StomsomiStartProduct=<file>    The first product providing ozone values for ozone interpolation. Use either the TOMSOMI and NCEP products or the atmosphericAuxdataPath to as source for ozone and air pressure.
                                  This is an optional source.

Parameter Options:
  -PatmosphericAuxDataPath=<string>    Path to the atmospheric auxiliary data directory. Use either this or tomsomiStartProduct, tomsomiEndProduct, ncepStartProduct and ncepEndProduct to use ozone and air pressure aux data for calculations. If the auxiliary data needed for interpolation not available in this path, the data will automatically downloaded.
  -PdemName=<string>                   The digital elevation model.
                                       Value must be one of 'Copernicus 90m Global DEM', 'Copernicus 30m Global DEM', 'GETASSE30'.
                                       Default value is 'Copernicus 90m Global DEM'.
  -Pelevation=<double>                 Used as fallback if elevation could not be taken from DEM.
                                       Valid interval is (0, 8500).
                                       Default value is '0'.
                                       Parameter unit is 'm'.
  -PoutputAsRrs=<boolean>              Reflectance values in the target product shall be either written as remote sensing or water leaving reflectances
                                       Default value is 'false'.
  -PoutputRtosa=<boolean>              Sets parameter 'outputRtosa' to <boolean>.
                                       Default value is 'false'.
  -Pozone=<double>                     The value used as ozone if not provided by auxiliary data
                                       Valid interval is (0, 1000).
                                       Default value is '330'.
                                       Parameter unit is 'DU'.
  -Ppress=<double>                     The surface air pressure at sea level if not provided by auxiliary data
                                       Valid interval is (800, 1040).
                                       Default value is '1000'.
                                       Parameter unit is 'hPa'.
  -Psalinity=<double>                  The value used as salinity for the scene
                                       Valid interval is (0.000028, 43).
                                       Default value is '35.0'.
                                       Parameter unit is 'PSU'.
  -Ptemperature=<double>               The value used as temperature for the scene
                                       Valid interval is (0.000111, 36).
                                       Default value is '15.0'.
                                       Parameter unit is 'C'.
  -PvalidPixelExpression=<string>      Defines the pixels which are valid for processing
                                       Default value is '!(l2_flags.LAND || rhot_865 > 0.25)'.
