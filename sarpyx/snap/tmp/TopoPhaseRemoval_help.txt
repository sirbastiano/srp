OPERATOR: TopoPhaseRemoval
----------------------------------------
Usage:
  gpt TopoPhaseRemoval [options] 

Description:
  Compute and subtract TOPO phase


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -PdemName=<string>                   The digital elevation model.
                                       Default value is 'SRTM 3Sec'.
  -PexternalDEMFile=<file>             Sets parameter 'externalDEMFile' to <file>.
  -PexternalDEMNoDataValue=<double>    Sets parameter 'externalDEMNoDataValue' to <double>.
                                       Default value is '0'.
  -PorbitDegree=<int>                  Degree of orbit interpolation polynomial
                                       Valid interval is (1, 10].
                                       Default value is '3'.
  -PoutputElevationBand=<boolean>      Output elevation band.
                                       Default value is 'false'.
  -PoutputLatLonBands=<boolean>        Output lat/lon bands.
                                       Default value is 'false'.
  -PoutputTopoPhaseBand=<boolean>      Output topographic phase band.
                                       Default value is 'false'.
  -PtileExtensionPercent=<string>      Define extension of tile for DEM simulation (optimization parameter).
                                       Default value is '100'.
