OPERATOR: AddLandCover
----------------------------------------
Usage:
  gpt AddLandCover [options] 

Description:
  Creates a land cover band


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PexternalFiles=<file,file,file,...>           The external landcover files.
  -PlandCoverNames=<string,string,string,...>    The land cover model.
                                                 Default value is 'AAFC Canada Sand Pct'.
  -PresamplingMethod=<string>                    Sets parameter 'resamplingMethod' to <string>.
                                                 Default value is 'NEAREST_NEIGHBOUR'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>AddLandCover</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <landCoverNames>string,string,string,...</landCoverNames>
        <externalFiles>file,file,file,...</externalFiles>
        <resamplingMethod>string</resamplingMethod>
      </parameters>
    </node>
  </graph>
