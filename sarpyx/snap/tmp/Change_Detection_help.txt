OPERATOR: Change-Detection
----------------------------------------
Usage:
  gpt Change-Detection [options] 

Description:
  Log Ratio Change Detection.


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PincludeSourceBands=<boolean>              Include source bands
                                              Default value is 'false'.
  -PmaskLowerThreshold=<float>                Mask lower threshold
                                              Default value is '-2.0'.
  -PmaskUpperThreshold=<float>                Mask upper threshold
                                              Default value is '2.0'.
  -PoutputDifference=<boolean>                Output Difference
                                              Default value is 'false'.
  -PoutputLogRatio=<boolean>                  Output Log Ratio
                                              Default value is 'true'.
  -PoutputNormalizedRatio=<boolean>           Output Normalized Ratio
                                              Default value is 'false'.
  -PoutputRatio=<boolean>                     Output Ratio
                                              Default value is 'false'.
  -PsourceBands=<string,string,string,...>    The list of source bands.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>

      <operator>Change-Detection</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <sourceBands>string,string,string,...</sourceBands>
        <maskUpperThreshold>float</maskUpperThreshold>
        <maskLowerThreshold>float</maskLowerThreshold>
        <includeSourceBands>boolean</includeSourceBands>
        <outputDifference>boolean</outputDifference>
        <outputRatio>boolean</outputRatio>
        <outputLogRatio>boolean</outputLogRatio>
        <outputNormalizedRatio>boolean</outputNormalizedRatio>
      </parameters>
    </node>
  </graph>
