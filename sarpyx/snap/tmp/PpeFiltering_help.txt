OPERATOR: PpeFiltering
----------------------------------------
Usage:
  gpt PpeFiltering [options] 

Description:
  Performs Prompt Particle Event (PPE) filtering on OLCI L1B


Source Options:
  -Ssource=<file>    OLCI L1B source product
                     Valid product types must match 'OL_1_E(F|R)R'.
                     This is a mandatory source.

Parameter Options:
  -PcutOff=<double>             Minimum threshold to differentiate with the neighboring pixels.
                                Default value is '0.7'.
  -PnumberOfMAD=<double>        Multiplier of MAD (Median Absolute Deviation) used for the threshold.
                                Default value is '10'.
  -PvalidExpression=<string>    An expression to filter which pixel are considered.
                                Default value is 'not quality_flags_land'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>PpeFiltering</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <cutOff>double</cutOff>
        <numberOfMAD>double</numberOfMAD>
        <validExpression>string</validExpression>
      </parameters>
    </node>
  </graph>
