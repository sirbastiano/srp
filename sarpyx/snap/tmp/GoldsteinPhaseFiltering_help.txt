OPERATOR: GoldsteinPhaseFiltering
----------------------------------------
Usage:
  gpt GoldsteinPhaseFiltering [options] 

Description:
  Phase Filtering


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -Palpha=<double>                 adaptive filter exponent
                                   Valid interval is (0, 1].
                                   Default value is '1.0'.
  -PcoherenceThreshold=<double>    The coherence threshold
                                   Valid interval is [0, 1].
                                   Default value is '0.2'.
  -PFFTSizeString=<string>         Sets parameter 'FFTSizeString' to <string>.
                                   Value must be one of '32', '64', '128', '256'.
                                   Default value is '64'.
  -PuseCoherenceMask=<boolean>     Use coherence mask
                                   Default value is 'false'.
  -PwindowSizeString=<string>      Sets parameter 'windowSizeString' to <string>.
                                   Value must be one of '3', '5', '7'.
                                   Default value is '3'.

Graph XML Format:
  <graph id="someGraphId">

        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <alpha>double</alpha>
        <FFTSizeString>string</FFTSizeString>
        <windowSizeString>string</windowSizeString>
        <useCoherenceMask>boolean</useCoherenceMask>
        <coherenceThreshold>double</coherenceThreshold>
      </parameters>
    </node>
  </graph>
