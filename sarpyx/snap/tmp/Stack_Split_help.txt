OPERATOR: Stack-Split
----------------------------------------
Usage:
  gpt Stack-Split [options] 

Description:
  Writes all bands to files.


Source Options:
  -Ssource=<file>    The source product to be written.
                     This is a mandatory source.

Parameter Options:
  -PformatName=<string>    The name of the output file format.
                           Default value is 'BEAM-DIMAP'.
  -PtargetFolder=<file>    The output folder to which the data product is written.
                           Default value is 'target'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Stack-Split</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <targetFolder>file</targetFolder>
        <formatName>string</formatName>
      </parameters>
    </node>
  </graph>
