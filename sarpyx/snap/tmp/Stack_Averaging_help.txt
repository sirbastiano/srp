OPERATOR: Stack-Averaging
----------------------------------------
Usage:
  gpt Stack-Averaging [options] 

Description:
  Averaging multi-temporal images


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -Pstatistic=<string>    Sets parameter 'statistic' to <string>.
                          Value must be one of 'Mean Average', 'Minimum', 'Maximum', 'Standard Deviation', 'Coefficient of Variation'.
                          Default value is 'Mean Average'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Stack-Averaging</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <statistic>string</statistic>
      </parameters>
    </node>
  </graph>
