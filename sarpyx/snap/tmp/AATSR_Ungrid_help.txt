OPERATOR: AATSR.Ungrid
----------------------------------------
Usage:
  gpt AATSR.Ungrid [options] 

Description:
  Ungrids (A)ATSR L1B products and extracts geolocation and pixel field of view data.


Source Options:
  -SsourceProduct=<file>    (A)ATSR-1/2 source product (Envisat *.N1 format)
                            This is a mandatory source.

Parameter Options:
  -PcornerReferenceFlag=<boolean>     Choose the pixel coordinate reference point for use in the output file. 
                                      Check for Corner (default), un-check for Centre.
                                      Default value is 'true'.
  -PL1BCharacterisationFile=<file>    L1B characterisation file is needed to specify first forward pixel and first nadir pixel
  -PtopographicFlag=<boolean>         Option to apply topographic corrections to tie points
                                      Default value is 'false'.
  -PtopographyHomogenity=<double>     Distance (image coordinates) pixel can be from tie-point to have topo correction applied
                                      Default value is '0.05'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>AATSR.Ungrid</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <L1BCharacterisationFile>file</L1BCharacterisationFile>
        <cornerReferenceFlag>boolean</cornerReferenceFlag>
        <topographicFlag>boolean</topographicFlag>
        <topographyHomogenity>double</topographyHomogenity>
      </parameters>
    </node>
  </graph>
