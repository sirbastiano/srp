OPERATOR: CoregistrationOp
----------------------------------------
Usage:
  gpt CoregistrationOp [options] 

Description:
  Coregisters two rasters, not considering their location


Source Options:
  -SsourceProduct.1=<file>    The source product which serves as slave.
                              This is a mandatory source.
  -SsourceProduct=<file>      The source product which serves as master.
                              This is a mandatory source.

Parameter Options:
  -Piterations=<int>             The number of interations for each level and for each radius.
                                 Default value is '2'.
  -Plevels=<int>                 The number of levels to process the images.
                                 Default value is '6'.
  -PmasterSourceBand=<string>    The master product band
  -Pradius=<string>              The radius integer values splitted by comma.
                                 Default value is '32, 28, 24, 20, 16, 12, 8'.
  -Prank=<int>                   Value used to compute the rank.
                                 Default value is '4'.
  -PslaveSourceBand=<string>     The slave product band

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>CoregistrationOp</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
        <sourceProduct.1>${sourceProduct.1}</sourceProduct.1>
      </sources>
      <parameters>
        <masterSourceBand>string</masterSourceBand>
        <slaveSourceBand>string</slaveSourceBand>
        <levels>int</levels>
        <rank>int</rank>
        <iterations>int</iterations>
        <radius>string</radius>
      </parameters>
    </node>
  </graph>
