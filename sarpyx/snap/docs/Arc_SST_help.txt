OPERATOR: Arc.SST
----------------------------------------
Usage:
  gpt Arc.SST [options] 

Description:
  Computes sea surface temperature (SST) from (A)ATSR and SLSTR products.


Source Options:
  -Ssource=<file>    The path of the (A)ATSR or SLSTR source product
                     This is a mandatory source.

Parameter Options:
  -Pasdi=<boolean>                      Enables/disables generation of ATSR Saharan Dust Index
                                        Default value is 'true'.
  -PasdiCoefficientsFile=<arcFiles>     Coefficient file for ASDI
                                        Value must be one of 'ASDI_ATSR1', 'ASDI_ATSR2', 'ASDI_AATSR'.
                                        Default value is 'ASDI_AATSR'.
  -PasdiMaskExpression=<string>         ROI-mask used for the ASDI
  -Pdual=<boolean>                      Enables/disables generation of the dual-view SST
                                        Default value is 'true'.
  -PdualCoefficientsFile=<arcFiles>     Coefficient file for the dual-view SST
                                        Value must be one of 'ARC_D2_ATSR1', 'ARC_D2_ATSR2', 'ARC_D2_AATSR', 'ARC_D2_SLSTR', 'ARC_D3_ATSR1', 'ARC_D3_ATSR2', 'ARC_D3_AATSR', 'ARC_D3_SLSTR'.
                                        Default value is 'ARC_D2_AATSR'.
  -PdualMaskExpression=<string>         ROI-mask used for the dual-view SST
  -PinvalidSstValue=<float>             Value used to fill invalid SST pixels
                                        Default value is '-999.0f'.
  -Pnadir=<boolean>                     Enables/disables generation of the nadir-view SST
                                        Default value is 'true'.
  -PnadirCoefficientsFile=<arcFiles>    Coefficient file for the nadir-view SST
                                        Value must be one of 'ARC_N2_ATSR1', 'ARC_N2_ATSR2', 'ARC_N2_AATSR', 'ARC_N2_SLSTR', 'ARC_N3_ATSR1', 'ARC_N3_ATSR2', 'ARC_N3_AATSR', 'ARC_N3_SLSTR'.
                                        Default value is 'ARC_N2_AATSR'.
  -PnadirMaskExpression=<string>        ROI-mask used for the nadir-view SST
  -PtcwvExpression=<string>             TCWV value to use in SST retrieval
                                        Default value is '30.0'.
