OPERATOR: Biophysical10mOp
----------------------------------------
Usage:
  gpt Biophysical10mOp [options] 

Description:
  The 'Biophysical Processor' operator retrieves LAI from atmospherically corrected Sentinel-2 products


Source Options:
  -Ssource=<file>    The source product.
                     This is a mandatory source.

Parameter Options:
  -PcomputeFapar=<boolean>     Compute FAPAR (Fraction of Absorbed Photosynthetically Active Radiation)
                               Default value is 'true'.
  -PcomputeFcover=<boolean>    Compute FVC (Fraction of Vegetation Cover)
                               Default value is 'true'.
  -PcomputeLAI=<boolean>       Compute LAI (Leaf Area Index)
                               Default value is 'true'.
  -Psensor=<string>            Sensor
                               Value must be one of 'S2A_10m', 'S2B_10m', 'S2C_10m'.
                               Default value is 'S2A_10m'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Biophysical10mOp</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <sensor>string</sensor>
        <computeLAI>boolean</computeLAI>
        <computeFapar>boolean</computeFapar>
        <computeFcover>boolean</computeFcover>
      </parameters>
    </node>
  </graph>
