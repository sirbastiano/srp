OPERATOR: Rad2Refl
----------------------------------------
Usage:
  gpt Rad2Refl [options] 

Description:
  Provides conversion from radiances to reflectances or backwards.


Source Options:
  -Ssource=<file>    The source product.
                     This is a mandatory source.

Parameter Options:
  -PconversionMode=<string>            Conversion mode: from rad to refl, or backwards
                                       Value must be one of 'RAD_TO_REFL', 'REFL_TO_RAD'.
                                       Default value is 'RAD_TO_REFL'.
  -PcopyFlagBandsAndMasks=<boolean>    If set, all flag bands and masks from source product are written to target product
                                       Default value is 'false'.
  -PcopyNonSpectralBands=<boolean>     If set, all other non-spectral bands from source product are written to target product
                                       Default value is 'false'.
  -PcopyTiePointGrids=<boolean>        If set, all tie point grids from source product are written to target product
                                       Default value is 'false'.
  -Psensor=<sensor>                    The sensor
                                       Value must be one of 'MERIS', 'OLCI', 'SLSTR_500m'.
                                       Default value is 'OLCI'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>

        <source>${source}</source>

        <sensor>sensor</sensor>

        <copyTiePointGrids>boolean</copyTiePointGrids>
        <copyFlagBandsAndMasks>boolean</copyFlagBandsAndMasks>
        <copyNonSpectralBands>boolean</copyNonSpectralBands>
      </parameters>
    </node>
  </graph>
