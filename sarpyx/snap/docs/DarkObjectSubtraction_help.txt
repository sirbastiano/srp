OPERATOR: DarkObjectSubtraction
----------------------------------------
Usage:
  gpt DarkObjectSubtraction [options] 

Description:
  Performs dark object subtraction for spectral bands in source product.


Source Options:
  -SsourceProduct=<file>    Source product containing spectral bands.
                            This is a mandatory source.

Parameter Options:
  -PhistogramMinimumPercentile=<int>              Percentile of minimum in image data in percent (the number means how many percent of the image data are lower than detected minimum.
                                                  Value must be one of '0', '1', '5'.
  -PmaskExpression=<string>                       Mask expression defining search area for dark object.
  -PsourceBandNames=<string,string,string,...>    Bands to be copied to the target. DOS will be applied on spectral bands only.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>DarkObjectSubtraction</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <sourceBandNames>string,string,string,...</sourceBandNames>
        <maskExpression>string</maskExpression>
        <histogramMinimumPercentile>int</histogramMinimumPercentile>
      </parameters>
    </node>
  </graph>
