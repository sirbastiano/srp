OPERATOR: StampsExport
----------------------------------------
Usage:
  gpt StampsExport [options] 

Description:
  Export data for StaMPS processing


Parameter Options:
  -PpsiFormat=<boolean>    Format for PSI or SBAS
                           Default value is 'true'.
  -PtargetFolder=<file>    The output folder to which the data product is written.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>StampsExport</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <targetFolder>file</targetFolder>
        <psiFormat>boolean</psiFormat>
      </parameters>
    </node>
  </graph>
