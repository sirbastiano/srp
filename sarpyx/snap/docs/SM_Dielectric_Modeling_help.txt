OPERATOR: SM-Dielectric-Modeling
----------------------------------------
Usage:
  gpt SM-Dielectric-Modeling [options] 

Description:
  Performs SM inversion using dielectric model


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -PeffectiveSoilTemperature=<double>    Effective soil temperature
                                         Valid interval is [-50.0, 50.0].
                                         Default value is '18.0'.
                                         Parameter unit is 'Celsius'.
  -PmaxSM=<double>                       Maximum soil moisture value
                                         Valid interval is (0.0, 1.0].
                                         Default value is '0.55'.
                                         Parameter unit is 'm^3/m^3'.
  -PminSM=<double>                       Minimum soil moisture value
                                         Valid interval is [0.0, 1.0).
                                         Default value is '0.0'.
                                         Parameter unit is 'm^3/m^3'.
  -PmodelToUse=<string>                  Choice of dielectric models for SM inversion
                                         Value must be one of 'Hallikainen', 'Mironov'.
                                         Default value is 'Hallikainen'.
  -PoutputLandCover=<boolean>            Optional LandCover in output
                                         Default value is 'true'.
  -PoutputRDC=<boolean>                  Optional RDC in output
                                         Default value is 'true'.

      </parameters>
