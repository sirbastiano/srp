OPERATOR: SetNoDataValue
----------------------------------------
Usage:
  gpt SetNoDataValue [options] 

Description:
  Set NoDataValueUsed flag and NoDataValue for all bands


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PnoDataValue=<double>         Sets parameter 'noDataValue' to <double>.
                                 Default value is '0.0'.
  -PnoDataValueUsed=<boolean>    Sets parameter 'noDataValueUsed' to <boolean>.
                                 Default value is 'true'.

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>SetNoDataValue</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <noDataValueUsed>boolean</noDataValueUsed>
        <noDataValue>double</noDataValue>
      </parameters>
    </node>
  </graph>
