OPERATOR: ArviOp
----------------------------------------
Usage:
  gpt ArviOp [options] 

Description:
  Atmospherically Resistant Vegetation Index belongs to a family of indices with built-in atmospheric corrections.


Source Options:
  -Ssource=<file>    The source product.
                     This is a mandatory source.

Parameter Options:
  -PblueFactor=<float>         The value of the BLUE source band is multiplied by this value.
                               Default value is '1.0F'.
  -PblueSourceBand=<string>    The blue band for the ARVI computation. If not provided, the operator will try to find the best fitting band.
  -Pdownsampling=<string>      The method used for aggregation (downsampling to a coarser resolution).
                               Value must be one of 'First', 'Min', 'Max', 'Mean', 'Median'.
                               Default value is 'First'.
  -PgammaParameter=<float>     The gamma parameter is like a weighting function that depends on the aerosol type
                               Default value is '1.0F'.
  -PnirFactor=<float>          The value of the NIR source band is multiplied by this value.
                               Default value is '1.0F'.
  -PnirSourceBand=<string>     The near-infrared band for the ARVI computation. If not provided, the operator will try to find the best fitting band.
  -PredFactor=<float>          The value of the RED source band is multiplied by this value.
                               Default value is '1.0F'.
  -PredSourceBand=<string>     The red band for the ARVI computation. If not provided, the operator will try to find the best fitting band.
  -PresampleType=<string>      If selected band s differ in size, the resample method used before computing the index
                               Value must be one of 'None', 'Lowest resolution', 'Highest resolution'.
                               Default value is 'None'.
  -Pupsampling=<string>        The method used for interpolation (upsampling to a finer resolution).
                               Value must be one of 'Nearest', 'Bilinear', 'Bicubic'.
                               Default value is 'Nearest'.
