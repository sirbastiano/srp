OPERATOR: ForestCoverChangeOp
----------------------------------------
Usage:
  gpt ForestCoverChangeOp [options] 

Description:
  Creates forest change masks out of two source products


Source Options:
  -SpreviousProduct=<file>    The source product to be modified.
                              This is a mandatory source.
  -SrecentProduct=<file>      The source product to be modified.
                              This is a mandatory source.

Parameter Options:
  -PcurrentProductSourceMaskFile=<file>           A binary raster file to be added as mask to the output product
  -PdegreesOfFreedom=<double>                     Degrees of freedom used for the Chi distribution trimming process
                                                  Default value is '3.3'.
  -PforestCoverPercentage=<float>                 Specifies the percentage of forest cover per segment
                                                  Default value is '95.0'.
  -PlandCoverMapIndices=<string>                  The indices of forest color from the new added land cover map
                                                  Default value is '40, 50, 60, 61, 62, 70, 71, 72, 80, 81, 82, 90, 100, 110, 160, 170'.
  -PlandCoverName=<string>                        Sets parameter 'landCoverName' to <string>.
                                                  Default value is 'CCILandCover-2015'.
  -PmergingCostCriterion=<string>                 The method to compute the region merging.
                                                  Value must be one of 'Spring', 'Baatz & Schape', 'Full Lamda Schedule'.
                                                  Default value is 'Spring'.
  -PpreviousProductSourceMaskFile=<file>          A binary raster file to be added as mask to the output product
  -PregionMergingCriterion=<string>               The method to check the region merging.
                                                  Value must be one of 'Best Fitting', 'Local Mutual Best Fitting'.
                                                  Default value is 'Local Mutual Best Fitting'.
  -PshapeWeight=<float>                           The shape weight.
                                                  Default value is '0.5'.
  -PspectralWeight=<float>                        The spectral weight.
                                                  Default value is '0.5'.
  -Pthreshold=<float>                             The threshold.
                                                  Default value is '5.0'.
  -PtotalIterationsForSecondSegmentation=<int>    The total number of iterations.
                                                  Default value is '10'.
