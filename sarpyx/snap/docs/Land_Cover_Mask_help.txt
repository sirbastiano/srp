OPERATOR: Land-Cover-Mask
----------------------------------------
Usage:
  gpt Land-Cover-Mask [options] 

Description:
  Perform a masking based on land cover classes


Source Options:
  -Ssource=<file>    Sets source 'source' to <filepath>.
                     This is a mandatory source.

Parameter Options:
  -PincludeOtherBands=<boolean>                Add other bands unmasked
                                               Default value is 'false'.
  -PlandCoverBand=<string>                     Land cover band
  -PsourceBands=<string,string,string,...>     The list of source bands.
  -PvalidLandCoverClasses=<int,int,int,...>    Land cover classes to include
  -PvalidPixelExpression=<string>              Valid pixel expression

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>Land-Cover-Mask</operator>
      <sources>
        <source>${source}</source>
      </sources>
      <parameters>
        <sourceBands>string,string,string,...</sourceBands>
        <landCoverBand>string</landCoverBand>
        <validLandCoverClasses>int,int,int,...</validLandCoverClasses>
        <validPixelExpression>string</validPixelExpression>
        <includeOtherBands>boolean</includeOtherBands>
      </parameters>
    </node>
  </graph>
