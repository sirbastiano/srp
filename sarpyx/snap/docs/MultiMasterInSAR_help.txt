OPERATOR: MultiMasterInSAR
----------------------------------------
Usage:
  gpt MultiMasterInSAR [options] 

Description:
  Multi-master InSAR processing


Source Options:
  -SsourceProduct=<file>    Sets source 'sourceProduct' to <filepath>.
                            This is a mandatory source.

Parameter Options:
  -PcohWindowAz=<int>                   Size of coherence estimation window in azimuth
                                        Default value is '10'.
  -PcohWindowRg=<int>                   Size of coherence estimation window in range
                                        Default value is '10'.
  -PincludeIncidenceAngle=<boolean>     Sets parameter 'includeIncidenceAngle' to <boolean>.
                                        Default value is 'true'.
  -PincludeLatLon=<boolean>             Sets parameter 'includeLatLon' to <boolean>.
                                        Default value is 'true'.
  -PincludeWavenumber=<boolean>         Sets parameter 'includeWavenumber' to <boolean>.
                                        Default value is 'true'.
  -PorbitDegree=<int>                   Degree of orbit (polynomial) interpolator
                                        Value must be one of '1', '2', '3', '4', '5'.
                                        Default value is '4'.
  -Ppairs=<string,string,string,...>    List of interferometric pairs

Graph XML Format:
  <graph id="someGraphId">
    <version>1.0</version>
    <node id="someNodeId">
      <operator>MultiMasterInSAR</operator>
      <sources>
        <sourceProduct>${sourceProduct}</sourceProduct>
      </sources>
      <parameters>
        <orbitDegree>int</orbitDegree>
        <pairs>string,string,string,...</pairs>
        <includeWavenumber>boolean</includeWavenumber>
        <includeIncidenceAngle>boolean</includeIncidenceAngle>
        <includeLatLon>boolean</includeLatLon>
        <cohWindowAz>int</cohWindowAz>
        <cohWindowRg>int</cohWindowRg>
      </parameters>
    </node>
  </graph>
